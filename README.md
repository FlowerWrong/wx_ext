# WxExt

微信接口之外的扩展gem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wx_ext', '~> 0.0.6'
```

And then execute:

    $ bundle

## Usage

使用请看spec/wx_ext/

## 测试

1. 修改 spec/wx_ext/weixin_spec.rb 文件中的账户密码
2. rspec spec/wx_ext/weixin_spec.rb
